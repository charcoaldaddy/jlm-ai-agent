## Development Setup

1. Install Rust
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2. Install Node.js
```bash
nvm install 20
```

3. Install pnpm
```bash
corepack enable && corepack prepare pnpm@8.15.0 --activate
```

4. Clone and setup
```bash
git clone https://github.com/charcoaldaddy/jlm-ai-agent.git
cd jlm-ai-agent
pnpm install
```

## Making Changes
